#网页搭建

用了一个早上的功夫，终于建成了。之前也想过人工搭建……算了，hexo是真的香！
来说说我搭建时我遇到的问题吧。

部署问题

在我部署到GitHub上时，出现ERROR Deployer not found: git报错，执行npm install --save hexo-deployer-git即可
还有这种报错error: failed to push some refs to 'xxxx'大概率是没有按照操作文件命名仓库的原因，按照要求改名成
“用户名”.github.io即可

排版问题

在对_config.yml这个文件进行修改的时候，预览发现中文标题出现乱码。这是由于_config.yml这个文档的编码方式不支持中文造成的，即便是把language属性改成zh_CN也没用。这个时候要将_config.yml文件编码改成UTF-8就迎刃而解了。

其他杂谈

刚开始还不适应，用了一会儿就深刻体会到命令行操作的效率与简洁；如你所见，还有好多地方没有设置与更改，以后再来完善吧。
仓促建站，也没有截屏，以后会注意的。
现在还在找一个好看的主题，目前就将就着用吧。
那就，先这样吧。
